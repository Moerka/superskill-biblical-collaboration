# Superskill: Biblical Collaboration & Unified Automation

This superskill integrates all available Manus capabilities into a single, collaborative framework. It automatically enrolls collaborators into a shared task environment, assigns roles based on Biblical principles of shared labor and community, and tracks contributions via a shared points system.

## Core Capabilities Integrated

This superskill seamlessly orchestrates the following domains:
1. **Automation & Development**: Master Automation Engine, GitHub Gem Seeker
2. **Content & Media**: BGM Prompter, Video Generator, Excel Generator
3. **Data & Research**: Stock Analysis, SimilarWeb Analytics, Internet Skill Finder
4. **Productivity**: GWS Best Practices

## The Collaborative Biblical Task System

When a task is initiated using this superskill, the following workflow MUST be executed:

### Phase 1: Automatic Enrollment & Role Assignment
All available collaborators (agents, users, or sub-routines) are automatically enrolled in the task. Roles are assigned based on the Biblical concept of the "Body of Christ" (1 Corinthians 12:12-27) — many parts, one body, each with a specific function.

### Phase 2: Task Execution & Contribution
Collaborators execute their assigned functions using the integrated skills. The system monitors contributions and ensures that no single entity bears the entire burden (Galatians 6:2 - "Carry each other's burdens").

### Phase 3: Shared Points & Rewards
Upon task completion or reaching milestones, points (or rewards) are distributed among all collaborators. The distribution follows the principle of shared harvest (Proverbs 11:25 - "A generous person will prosper; whoever refreshes others will be refreshed").

## Usage Instructions

To initiate a collaborative task, use the provided orchestration script:

```bash
python /home/ubuntu/skills/superskill-biblical-collaboration/scripts/orchestrate_collaboration.py --task "Task Description"
```
